# Claude API活用事例

## 概要
問い合わせ文、報告内容、案件情報などをClaude APIで要約、分類、回答下書きに変換する改善パターン。

## 課題
文章を読む、要約する、返信文を考える作業に時間がかかっている。

## 改善案
AIに任せる範囲を、下書き、要約、分類、チェックに限定し、最終判断は人が行う設計にする。

## おすすめツール
Claude API、GAS、Webアプリ、LINE Messaging API。

## 効果
文章処理の初動が早くなり、担当者の判断時間を確保できる。

## ポートフォリオで見せるポイント
AIを丸投げではなく、人が確認できる業務フローに組み込める。
