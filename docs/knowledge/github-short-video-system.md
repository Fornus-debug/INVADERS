# GitHubリポジトリ収集からショート動画生成システム

## 概要
GitHubリポジトリを収集し、内容を要約してショート動画の台本や素材へ変換する改善事例。

## 課題
技術系コンテンツのネタ探し、要約、台本作成、投稿準備に時間がかかる。

## 改善案
GitHub APIで候補を集め、AIで見どころを抽出し、ショート動画用の構成へ変換する。

## おすすめツール
GitHub API、ChatGPT、Claude API、スプレッドシート、動画生成ツール。

## 効果
情報収集から投稿準備までの流れを短縮し、継続発信しやすくなる。

## ポートフォリオで見せるポイント
AIと外部APIを組み合わせた半自動コンテンツ生成の実績として見せられる。
