# 案件収集AI

## 概要
複数の案件情報を集め、条件に合う案件を抽出し、応募判断に必要な情報を整理する改善事例。

## 課題
案件探しに時間がかかり、単価、技術要件、納期、相性の比較が手作業になっていた。

## 改善案
案件情報を一覧化し、AIで要約、スコアリング、応募文の下書き作成まで行う。

## おすすめツール
ChatGPT、Claude API、Googleスプレッドシート、GAS。

## 効果
営業活動の初動を早くし、応募すべき案件の見落としを減らせる。

## ポートフォリオで見せるポイント
AIを使って売上につながる業務を短縮する提案力を示せる。
