# LINE連携事例

## 概要
顧客とのやり取りが多い業務で、LINEを受付、通知、一次対応の入口として活用する改善パターン。

## 課題
問い合わせ対応が属人化し、返信漏れや対応履歴の見落としが起きやすい。

## 改善案
LINE Messaging APIで受信内容を記録し、よくある質問は自動返信、個別判断が必要な内容は担当者へ通知する。

## おすすめツール
LINE Messaging API、Webhook、GAS、Googleスプレッドシート。

## 効果
顧客対応の初動を早め、対応履歴を残しやすくする。

## ポートフォリオで見せるポイント
AIエージェントや予約受付への拡張余地を説明しやすい。
