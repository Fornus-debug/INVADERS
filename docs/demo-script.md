# 営業デモ説明文

## 一言説明

カタカタ改善診断AIは、現場や中小企業に残っている毎回の手作業を見つけ、AIが改善候補・削減時間・見積もり目安まで出すWeb診断デモです。

## デモの流れ

1. カタカタくんのENTERボタンを押す
2. 業務説明を入力する
3. 必要に応じてPDF・スクショ・動画ファイル名を添付する
4. AI改善係長が診断する
5. 診断結果をコピーする
6. LINE相談につなげる

## 説明時の注意

現在は静的HTMLのMVPです。
ファイル本文の自動解析はまだ実装していません。
ただし、実際の案件ではPDF抽出、OCR、動画解析、LLM API連携まで拡張できます。

## 提案トーク

「いきなり大きなシステムを作る前に、まずどの作業がムダになっているかを診断する入口として使えます。」

「相談前に課題を整理できるので、GAS、n8n、LINE、RAG、スプレッドシート自動化などの提案につなげやすくなります。」

「カタカタくんのようなキャラクターを使うことで、AIや自動化に苦手意識がある方にも使ってもらいやすくできます。」
